# Global Superstore Data Cleaning and Preparation

## Project Overview

This project focuses on cleaning and preparing the Global Superstore dataset using Python and Pandas in Google Colab. The goal is to improve data quality by identifying and handling missing values, removing duplicate records, converting data types, and exporting a clean dataset ready for analysis and visualization.

## Objectives

* Load the Global Superstore dataset into Google Colab.
* Explore the dataset structure and data quality.
* Identify and handle missing values.
* Detect and remove duplicate rows.
* Convert columns to appropriate data types (e.g., datetime).
* Export the cleaned dataset as a new CSV file.

## Tools and Technologies

* Python
* Pandas
* Google Colab
* CSV Files

## Data Cleaning Steps

1. Imported the Global Superstore dataset.
2. Examined dataset dimensions, columns, and data types.
3. Checked for missing values across all columns.
4. Applied appropriate methods to handle missing data.
5. Identified and removed duplicate records.
6. Converted date columns into datetime format.
7. Validated the cleaned dataset.
8. Exported the final cleaned dataset to a new CSV file.

## Project Outcomes

* Improved data quality and consistency.
* Removed redundant and duplicate records.
* Standardized date formats for easier analysis.
* Prepared the dataset for further exploratory data analysis (EDA), visualization, and business intelligence tasks.
